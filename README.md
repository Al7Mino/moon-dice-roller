# Moon Dice Roller

![Logo Moon Dice Roller][logo]

[logo]: ./logo.png "Moon Dice Roller"

Moon Dice Roller est un bot Discord de lancer de dés.

## Comment l'utiliser ?

### Prérequis

Node.js v16.9


### Développement

Ce bot Discord utilise le [framework sapphire][sapphire] et est écrit en TypeScript.

```sh
pnpm start
```
